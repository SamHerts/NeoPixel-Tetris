# NeoPixel-Tetris

A self-playing Tetris bot designed for [Xiao ESP32-C6](https://www.seeedstudio.com/Seeed-Studio-XIAO-ESP32C6-p-5884.html) and [Seeed 6x10 RGB WS2812 matrix](https://www.seeedstudio.com/6x10-RGB-MATRIX-for-XIAO-p-5771.html). Procedurally generates Tetris pieces, analyzes the board state using a heuristic algorithm, and automatically places pieces in the optimal position.
